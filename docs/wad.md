## 1. Introdução <a name="c1"></a>

Atualmente, os alunos de ensino médio e superior são cobrados intensivamente por meio de avaliações, sejam elas de marcar uma opção correta ou discorrer sobre um assunto. Apesar do intenso preparo desses estudantes, após a correção da prova, muitos simplesmente deixam o assunto tão estudado de lado, guardando somente fragmentos dos conhecimentos que antes tinham. E se houvesse uma maneira de guardar esses ensinamentos de maneira digital com a possibilidade de revisá-los de maneira interativa utilizando cards. Para isso surgiu o Scaneio, uma plataforma web de revisão de assuntos diferentes por meio de cards gerados por IA a partir de uma prova já feita.

O objetivo com o desenvolvimento desse projeto, para o desenvolvedor é praticar conceitos arquiteturais e tecnologias para aprimoramento pessoal, sem utilizar Inteligência Artificial para gerar todo o projeto.
