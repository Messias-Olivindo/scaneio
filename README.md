# Scaneio 🔎

Aplicação Web para relembrar assuntos de prvas e nunca se esquecer dos assuntos que você vê durante o ensino médio, faculdade, concurso e muitos outros.

## Estrutura do projeto

Para deixar claro como o projeto funciona e para que serve a divisão de pastas do repositório, segue o esqueleto estrutural do projeto:

```tree
├── backend/      Backend clássico de uma aplicação web
├── docs/         Documentação técnica do projeto  
├── frontend/     Interface da aplicação web
├── worker/       Worker de processamento dos documentos 
└── README.md     Arquivo com as informações básicas do projeto
```
